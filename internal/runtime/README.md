# 运行时说明

本目录预留给生成的基础库使用的内部运行时辅助代码。当前目录刻意不包含 Go 代码；没有明确调用点、测试和 contract 之前，不应新增运行时实现。

允许进入本目录的代码必须同时满足：

- 只服务于基础库自身的生命周期、环境无关适配或测试可复用内部逻辑。
- 不导出公共 API，不隐藏全局 runtime，不在包初始化阶段启动后台任务。
- 不依赖 `x.go`、业务 provider 或重型可观测性 SDK。
- 不隐式读取生产密钥路径；所有敏感值必须来自显式配置并经过 redaction。
- 新增实现必须配套单元测试，并继续通过 `make boundary` 和 `make security`。
